# API Reference

## Configuration

### Configuration Input Methods

#### Configuration Override

You can specify input parameters via Config Override (link)

#### GitHub Action `with:`

You can specify input parameters via GitHub Action yaml section `with` as shown in the "Getting
Started" examples.

#### Command Line

You can specify input parameters via command line.

#### Environment Variables

You can specify input parameters via environment variables.

## Modes

```md
- hash (hash all folder contents)
- cache-push (push to cache)
- cache-pull (pull from cache)
- remote-cli-pre-build (sets up a repository, usually before a game-ci build)
- print-input (prints all input)
- cli-build (runs a cloud runner build)
- garbage-collect (runs garbage collection)
- list-resources (lists active resources)
- list-worfklow (lists running workflows)
- watch (follows logs of a running workflow)
- remote-cli-post-build (runs a cloud runner build)
```
